
# Personal Portfolio Website

A simple and responsive personal portfolio website built using **HTML** and **CSS**.  
This project showcases my profile, strengths, projects, and contact information through a clean two-page design.

## Features

- Responsive portfolio design
- Home page with:
  - About Me section
  - Skills/Strengths
  - Project showcase
  - Social media links
- Separate Contact page with:
  - Contact form
  - Navigation button back to Home
- Built completely using HTML and CSS

## Pages

### 1. Home Page
The main portfolio page that introduces me, my skills, and my projects.

### 2. Contact Page
A separate page where users can enter their details and message through a contact form.

## Technologies Used

- HTML5
- CSS3
